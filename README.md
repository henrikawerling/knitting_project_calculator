## Setup

1. **Clone the repository and navigate to the src directory.**

    ```bash
    cd knitting_project_calculator/src
    ```

2. **Run the program from the command line.**

    macOS:
    ```bash
    python3 main.py
    ```
    Windows:
    ```bash
    python main.py
    ```
